### NAME
    environment() - return the environment of an object


### SYNOPSIS
    object environment( object ob );


### DESCRIPTION
    Return  the  containing object (environment) of <ob>. If no argument is
    given, <ob> defaults to this_object().


### SEE ALSO
    this_object(3)
