### NAME
    disable_wizard() - remove wizard priveleges from an object


### SYNOPSIS
    void disable_wizard( void );


### DESCRIPTION
    The  opposite  of enable_wizard().  Disables wizard privileges from the
    current object.


### SEE ALSO
    enable_wizard(3), wizardp(3)
