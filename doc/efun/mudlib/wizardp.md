### NAME
    wizardp()  - determines if a given object had enable_wizard() performed
    in it


### SYNOPSIS
    int wizardp( object );


### DESCRIPTION
    Returns 1 if the arg had enable_wizard() performed on it.

    disable_wizard(3), enable_wizard(3)
